# BARREL RUN [OUTSIDE EDITION]
A great project using a javascript engine.

# Instructions:
- Start game in MainMenu scene in Scenes!
- Go through the menu and click play!
- Go through as many levels as you can!
- Don't lose!

# Movement:
- *W/Space* - To jump
- *A* - Move left
- *D* - Move right
